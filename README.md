
#  **aurora-ui**

![React](https://img.shields.io/badge/react-19-61dafb)
![TypeScript](https://img.shields.io/badge/typescript-5.4-blue)

component library for analytics dashboards.

---

### install

```bash
npm i aurora-ui
```

### usage

```jsx
import { StatCard } from "aurora-ui"

<StatCard title="CPU Usage" value="73%" trend="up" />
```

### components

* `StatCard`
* `ProgressRing`
* `DataGrid`
* `HeatMap`

### features

* light/dark themes included
* headless mode
* full CSS variable customization
